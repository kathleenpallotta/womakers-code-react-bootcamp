# WoMakersCode React Bootcamp

A WoMakersCode é uma comunidade brasileira, que tem como objetivo o empoderamento e fortalecimento da presença feminina na tecnologia. O React Bootcamp tem a missão de alavancar a capacitação de mulheres na área de desenvolvimento de software, abrindo portas para atuarem como desenvolvedoras front-end, utilizando React, através de uma capacitação alinhada com o mercado de trabalho.
